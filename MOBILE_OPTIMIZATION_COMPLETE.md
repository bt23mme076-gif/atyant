## 📐 Responsive Breakpoints

### Large Desktop (>1200px):
- Full sliders (450px wide)
- Large heading (3.5rem)

### Desktop (1024px - 1200px):
- Medium sliders (350px wide)
- Medium heading (3rem)

### Tablet (768px - 1024px):
- Small sliders (300px wide)
- Smaller heading (2.5rem)

### Mobile (480px - 768px):
- **No sliders**
- **Purple gradient**
- **Centered layout**
- **White text**
- Heading: 2.25rem

### Small Mobile (<480px):
- **Extra compact**
- Heading: 1.85rem
- Smaller stats
- Full-width button

## 🔧 Customization

### Change Mobile Background Color:

```css
@media (max-width: 768px) {
  .internship-hero-topmate {
    background: linear-gradient(180deg, #YOUR_COLOR_1 0%, #YOUR_COLOR_2 100%);
  }
}
```

### Change Button Style:

```css
@media (max-width: 768px) {
  .hero-cta-btn {
    background: #YOUR_COLOR;
    color: #YOUR_TEXT_COLOR;
  }
}
```
