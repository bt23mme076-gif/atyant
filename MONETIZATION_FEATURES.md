# Monetization Dashboard - Feature Overview

## 🎯 What You Can Do Now

### 1. Overview Tab
**Dashboard at a glance**
- 💰 Total Earnings (last 30 days)
- 🎯 Active Services count
- 📅 Upcoming Bookings
- ✅ Completed Sessions
- Quick action buttons

### 2. Services Tab
**Create & Manage Your Offerings**

#### Service Types:
1. **📞 1:1 Sessions**
   - Set duration (15, 30, 60 minutes)
   - Set price
   - Students book specific time slots
   - Example: "Career Guidance Call - 30 min - ₹999"

2. **📦 Packages**
   - Bundle multiple sessions
   - Discounted pricing
   - Track sessions used
   - Example: "3 Session Package - ₹2499 (Save 30%)"

3. **📄 Digital Products**
   - Sell PDFs, templates, guides
   - Instant delivery
   - One-time purchase
   - Example: "Resume Template Bundle - ₹499"

#### Service Management:
- ✏️ Edit service details
- 🗑️ Delete services
- 👁️ Toggle active/inactive
- 📊 View sales & revenue per service

### 3. Bookings Tab
**Manage Your Sessions**
- View all bookings
- Filter by status:
  - ⏳ Pending
  - 📅 Upcoming
  - ✅ Confirmed
  - ✔️ Completed
  - ❌ Cancelled
- See student details
- Add meeting links (Google Meet, Zoom)
- Update booking status

### 4. Availability Tab
**Set Your Schedule** (Coming Soon)
- Weekly schedule
- Block specific dates
- Set buffer time between sessions
- Timezone support

### 5. Earnings Tab
**Track Your Income**
- Total earnings
- Transaction count
- Booking breakdown by status
- Period-based filtering

## 🔄 User Flow

### Mentor Creates Service
1. Click "💰 Monetization Dashboard" from Mentor Dashboard
2. Go to "Services" tab
3. Click "+ Create Service"
4. Fill in details:
   - Choose type (session/package/product)
   - Enter title & description
   - Set price
   - Set duration (for sessions) or sessions count (for packages)
   - Toggle active status
5. Click "Create"
6. Service is now visible to students

### Student Books Service (Future)
1. Visit mentor profile
2. See available services
3. Click "Book Now"
4. Select date/time (for sessions)
5. Pay via Razorpay
6. Receive confirmation email
7. Get meeting link

### Mentor Manages Booking
1. Go to "Bookings" tab
2. See new booking notification
3. Confirm booking
4. Add meeting link
5. Conduct session
6. Mark as completed
7. Earnings updated automatically
